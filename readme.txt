== Description ==

Appeto is a powerful Persian online application generator for android, ios and pwa.
With this plugin you can connect your Wordpress and Woocommerce shop to your application that have been generated on appeto panel.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/appeto-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Learn more about how to connect plugin to appeto panel: <a href="https://appeto.ir/?p=7085" target="_blank">https://appeto.ir/?p=7085</a>
